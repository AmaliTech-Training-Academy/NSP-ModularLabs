#  Lab: UX Analysis & Redesign – Art.yale.edu

---

##  Activity Outline

###  Website Review
Visit the [Art.yale.edu](https://art.yale.edu) website and evaluate its user experience.

---

### 1.  Flaw Identification
- Analyze the website thoroughly  
- Identify **bad UX elements** across the interface  

---

### 2.  Goal Setting & Ideation

#### Define a UX Goal
Establish a clear **“Good UX” goal** for the interface. Example:
> *Increase conversion by 20% through improved trust and clarity*

#### Ideation
- Generate solution ideas using quick sketching techniques  
- Suggested method: **Crazy 8s** (or similar rapid ideation techniques)  

---

### 3.  Low-Fidelity Redesign
- Create a **low-fidelity wireframe or sketch** of the improved interface  
- Focus on:
  - Clarity  
  - Efficiency  
  - Established UX patterns  

---

##  Deliverables

-  Documented list of bad UX examples  
  - Include explanations of design intent or root causes  

-  A clear goal statement  
  - Include initial design concepts or sketches  

-  A revised digital wireframe  
  - Cover key screens or user flows

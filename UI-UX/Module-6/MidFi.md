
# Learning Objectives
The Pantry Pal is a collaborative mobile application designed for small, shared households (roommates, families) to manage their favorite recipes, plan weekly meals together, and auto-generate a shared shopping list.

## Activity Outline
1. Login to the system so that I can have access to the application
2. Reset my password so that I can have access to the system if I lose my password.
3. Recipe Creation
a. The process of adding a new recipe must be optimized for typing on a phone keyboard.
b. Multi-Step Input: Design this as a short, multi-step process (a wizard) rather than a single scrolling form to reduce cognitive load (e.g., Step 1: Name & Time; Step 2: Ingredients; Step 3: Steps).
c. Structured Input: Use smart fields or native mobile pickers for ingredients to quickly capture quantity, unit (e.g., a scroll wheel for cups/grams), and item name.
d. Image Upload: Include a clear placeholder/button for uploading a recipe photo using the device's camera.
4. Search and filter through the list of recipes to get the one I prefer.
5. Share to another user on the platform to have access to my recipe.
6. Logout of the system.

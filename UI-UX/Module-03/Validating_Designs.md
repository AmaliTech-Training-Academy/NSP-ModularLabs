
##  Learning Objectives

- Understand the importance of user research in validating design assumptions  
- Develop a user interview plan to gather feedback on a mobile app design  
- Craft effective interview questions to uncover user needs and behaviors  
- Conduct user interviews in a user-centered manner  
- Analyze interview data to validate or challenge design assumptions  

---

## Activity Outline

### 1. Design Assumption Selection
Choose a specific design assumption related to the mobile app being assessed.  
This could focus on:
- A feature  
- Navigation flow  
- Information presentation  

---

### 2. Target User Identification
Define the ideal user group for the mobile app.  
This will help guide participant recruitment for interviews.

---

### 3. Interview Question Development

Create open-ended, user-centered questions that:

####  Gather Background Information
- User demographics  
- App usage habits  
- General needs  

####  Explore the Design Assumption
- Questions directly related to the assumption  
- How users would interact with the design  

#### Uncover User Needs
- Pain points  
- Desired features or improvements  

---

### 4. Interview Script Development
Structure the interview flow:
- Introduction  
- Transition between topics  
- Closing remarks  

Ensure:
- Flexibility for follow-up questions  
- Natural conversation flow  

---

### 5. Recruiting Participants
Identify users matching your target group:
- Online recruitment platforms  
- Student networks  
- Relevant communities (based on app audience)  

---

### 6. Conducting Interviews

####  Preparation
- Set up a comfortable environment  
- Ensure recording tools work properly  

####  Introduction & Consent
- Explain purpose of the interview  
- Obtain informed consent  

####  Interview Execution
- Follow your script  
- Ask follow-up questions where necessary  
- Stay user-centered and flexible  

---

### 7. Data Analysis

Analyze collected data using:

####  Thematic Analysis
- Identify recurring themes and patterns  

####  Affinity Mapping
- Group similar quotes and observations  
- Organize insights into themes  

---

### 8. Assumption Validation
- Determine if the design assumption is supported by user feedback  
- If not, identify improvements based on insights  

---

## Additional Tips

- Practice active listening during interviews  
- Record sessions (with consent) for accurate analysis  
- Use role-playing to practice interview techniques before real interviews
